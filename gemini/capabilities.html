What Mantis Can Do

Voice Interaction

With microphone and speakers, the robot can interact with the user through voice commands, making it intuitive and easy to use.

AI Reasoning

With the latest AI models the robot can understand voice commands and respond or perform tasks depending on the request

Object Recognition

Equipped with computer vision for real-time object detection and an autonomous mode for searching a target.

VR Headset Control

Can be controlled via a VR headset, providing an immersive experience for remote operations.

Robotic Arm

Equipped with a 6-DOF arm for tasks like picking/moving up objects. It has two modes of movement: direct and inverse kinematics.

Lidar Sensor

Equipped with a lidar sensor enabling autonomous navigation, obstacle avoidance and distance-to-target assessment.

Standalone System

All the capabilities (except voice interaction) are available without any internet connection or external hardware, ensuring functionality in any location.
