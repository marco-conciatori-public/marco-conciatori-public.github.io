Getting Started: Setup Guide for Jade

  1. 1. Prepare the Raspberry Pi

    Flash Ubuntu 18.04 onto an SD card and insert it into the Raspberry Pi. Ensure you have SSH access configured. 

    sudo apt update && sudo apt upgrade

  2. 2. Install ROS Melodic

    Follow the official ROS Melodic installation guide for Ubuntu. This is crucial for robot communication. 

    sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
sudo apt update
sudo apt install ros-melodic-desktop-full

  3. 3. Connect Hardware Components

    Carefully connect the motor drivers, camera, lidar, and other sensors to the Raspberry Pi's GPIO pins and USB ports according to the wiring diagram.

  4. 4. Clone Robot Software Repository

    Clone the robot's software repository from GitHub and compile the ROS packages. 

    cd ~/catkin_ws/src
git clone https://github.com/marcorobot/robot_software.git
cd ~/catkin_ws && catkin_make

  5. 5. Run Initial Tests

    Start the core ROS nodes and verify all sensors are functioning correctly and motors respond to commands. 

    roslaunch robot_bringup robot_core.launch
