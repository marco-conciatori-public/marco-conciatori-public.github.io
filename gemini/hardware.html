Mantis' Hardware Components

From RDK X3 Robot (Yahboom)

RDK X3 Development Board

The main processor. It came already set up with Ubuntu 18.04.

ROS Expansion Board

Features the main switch, connections for wheel and arm motors, and distributes power from the battery to all other connected components.

USB HUB Board

Increases the number and type of ports available for connecting to the RDK X3 Development Board.

RGB Cooling Hat

Includes a fan for cooling board stack (RDK X3, ROS Expansion, USB HUB) and some lights controllable via the PS2 controller.

OLED Screen

Used for displaying text, which is useful for checking the robot's state without a computer connection.

IMX219 MIPI Camera

A camera that connects to the RDK X3 Development Board with a wide, flat ribbon cable.

MS200 Lidar

Senses the direction and distance of objects in the horizontal plane. It comes with a dedicated adapter board.
Note: In the current setup, the lidar is partially occluded and can only see in front and to the sides. Mounting it higher would obstruct its view of lower obstacles.

Wireless PS2-like Controller

Used for manual control of the robot; its behavior can be customized via software.

Battery

Powers the ROS Expansion Board and all connected components. It can last from 30 minutes to a few hours depending on usage and has a specific charger.

From DOFBOT - Jetson NANO (Yahboom)

Robotic Arm

A pre-assembled robotic arm with 6 Degrees of Freedom (DOF).

Jetson Nano Board

The arm's main computer, which is currently used only for voice interaction. It's configured to boot from USB instead of a TF Card and includes a wireless network card.

U Disk (64 GB)

A metal USB drive that contains the OS and memory for the Jetson Nano.

USB Camera

A 0.3 MP USB camera with a support for attaching it to the top of the robotic arm.

From Mechanum Wheel Chassis (Yahboom)

Chassis

A large chassis configured to host the robotic arm, featuring a "pendulous suspension" to ensure all four mechanum wheels maintain ground contact for correct operation.

Wheels and Motors

Includes larger mechanum wheels and motors. These motors have a higher reduction ratio, making them slower but capable of carrying more weight.

Separately Acquired Components

Coral USB Accelerator (Google)

An Edge TPU used to speed up AI computations. It allows for local execution of YOLO models for object recognition and the autonomous vision agent, accelerating frame processing by nearly 10 times. It's possible this could be replaced by the Jetson Nano.

Battery for the Jetson Nano (Zeee)

A Lipo battery (3S 5200mAh with Deans T Plug) that powers all devices on the Robotic Arm Expansion Board (Jetson Nano, microphone, speakers, USB Camera). Requires a compatible charger like the WANGCL IMAX B6AC.

ReSpeaker Mic Array v2.0 (Seeed Studio)

A microphone array with sound filtering capabilities and Direction of Arrival (DoA) computation. It is connected to the Jetson Nano for voice interaction but must be connected to the main board for the sound-following agent.

Speakers

A pair of standard speakers (casings removed) connected to the ReSpeaker Mic Array via a 3.5mm audio jack.

Other Customizations

Various custom parts including:
- A 128 GB TF Card for the RDK X3.
- Specific cables and a 40-pin extension for easier access to the RDK X3's GPIO.
- Physical LEDs and switches connected to the GPIO for controller-free operation of autonomous modes.
- Extra screws and pillars for mounting.
- A protective U-shaped metal mesh (from an old PC case) for the Jetson Nano.
- Sturdy cardboard for mounting the speakers.
