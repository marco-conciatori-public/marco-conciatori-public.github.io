Quick Start Installation

1

Clone Repository

git clone https://github.com/yourusername/techflow-robot.git
cd techflow-robot
2

Install Dependencies

./scripts/install_dependencies.sh
rosdep install --from-paths src --ignore-src -r -y
3

Build Workspace

colcon build --symlink-install
source install/setup.bash
4

Launch Robot

ros2 launch techflow_robot robot.launch.py
# In another terminal:
ros2 launch techflow_robot navigation.launch.py
