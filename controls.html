Mantis Controller Guide

There are many actions and commands that the robot can perform, far more than the buttons on the PS2 controller. To address this problem the controls are divided into different modes, each with its own set of functionalities. For example, in User controlled mode (wheels) the joysticks control the movement of the robot, while in User controlled mode (arm - forward kinematics) the same joysticks control the movements of the robotic arm. The General controls keys are always active, regardless of the current mode.
